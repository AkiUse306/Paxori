# Paxori

Paxori is a modern, cross-platform file transfer application designed to make sharing files between devices fast, secure, and effortless. Whether you're transferring a single document or an entire project, Paxori provides a clean, intuitive experience without unnecessary complexity.

Built with performance, reliability, and simplicity in mind, Paxori works seamlessly across Windows, macOS, and Linux, allowing devices on different operating systems to communicate with each other without additional configuration. Its lightweight architecture minimizes resource usage while maintaining high transfer speeds and dependable connections.

# Architecture

Paxori is built using a modern, modular architecture that separates the user interface, networking, and transfer engine into independent components. This design makes the application easier to maintain, extend, and optimize while ensuring a consistent experience across all supported operating systems.

## Core components include:

### User Interface – A responsive desktop application that provides a simple and intuitive experience.
### Transfer Engine – Responsible for discovering devices, managing connections, and transferring files efficiently.
### Networking Layer – Handles communication between devices with an emphasis on reliability and security.
### Platform Integration – Provides native installation, system integration, and operating-system-specific functionality for Windows, macOS, and Linux.

This modular approach allows future improvements and new features to be added without affecting the stability of the rest of the application.

# Installation
## Windows

Paxori provides multiple installation options for Windows.

MSI Installer (Recommended)
Installs Paxori using the standard Windows Installer with desktop shortcuts, Start Menu integration, and automatic uninstallation support.
PowerShell Installer (install.ps1)
Downloads and installs the latest version of Paxori automatically.
Batch Support Script (.bat)
Provides a simple command-line method for launching or assisting with installation in environments where PowerShell may not be preferred.

## MacOS

Paxori supports both graphical and terminal-based installation.

DMG Package (Recommended)
Open the downloaded .dmg file and drag Paxori into the Applications folder.
Shell Installer (install.sh)
Installs Paxori directly from the terminal for users who prefer command-line installation.

## Linux

Paxori supports the most common Linux distribution formats.

Debian Package (.deb)
For Debian, Ubuntu, Linux Mint, Pop!_OS, and other Debian-based distributions.
AppImage
A portable package that runs without requiring installation.
Shell Installer (install.sh)
Installs Paxori automatically and configures desktop integration where supported.
