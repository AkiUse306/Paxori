Installation

Install Paxori in seconds from GitHub releases.

Windows

Install directly from GitHub with a single PowerShell command.

powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/AkiUse306/Paxori/main/install.ps1 | iex"

This installer downloads the release ZIP from GitHub and registers uninstall metadata.

macOS / Linux

Install for your user or system-wide using the shell installer.

curl -fsSL https://raw.githubusercontent.com/AkiUse306/Paxori/main/install.sh | bash
sudo curl -fsSL https://raw.githubusercontent.com/AkiUse306/Paxori/main/install.sh | sudo bash --global

Global install places Paxori under /usr/local; user install stays in your home directory.

Windows CLI

Install from Command Prompt with optional elevated system install.

install.bat
install.bat --global

Global install writes Paxori into Program Files and supports running from anywhere after PATH setup.
